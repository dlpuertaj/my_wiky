# flutter_launcher_icons

> Created: 2026-04-15

A Flutter package that automates generating all platform-specific launcher icon sizes from a single source image.

Takes a 1024×1024 PNG and outputs every required Android (and optionally iOS) icon size with a single command.

## Usage

Add the package to `pubspec.yaml`, then run:

```bash
flutter pub get
dart run flutter_launcher_icons
```

Output lands in `android/app/src/main/res/` for Android.
